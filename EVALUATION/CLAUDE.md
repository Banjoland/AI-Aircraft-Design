## Evaluation

You will evaluate the simulated performance of each model to determine how well its performance met the design specification.  You will compute a total cost function using the methods described in the document COST FUNCTION.md.  You will sum the total cost.  The design with the lowest cost should be considered the best design and used for the basis for the next design.

The evaluation task should use subagents defined in the AGENTS directory

You must determine if you have the requisite agents, tools and skills to perform each task.  If you do not, you must create a new agent, tool and/or task.

For each new agent, tool or task you should create a README.md file for human readers to understand what the agent does and how it does it. Each agent should have a test directory and test software to test the functionality of the agent, tools and/or tasks.  These can include a copy of prototype models to perform the test on.

You can add to this file as you refine this process

You must add the following capabilities:

1. determine a way to measure the curvature of your fuselage.
2. report on the maximum curvature of your fuselage
3. be able to analyze high pressure regions contributing to high drag due to a lack of streamlined profiles
