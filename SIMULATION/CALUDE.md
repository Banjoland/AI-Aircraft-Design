## Simulation

You will use the program Aerovsp.exe to evaluate candidate aircraft performance.  The minimum analysis you must perform is described in the document SIM_SPEC.  Guidelines:

1. Use subagents to perform this task
2. Agents should be put in the AGENTS directory
3. Agents should have access to tools and tasks
4. You must determine if you have the requisite agents, tools and skills to perform each task.  If you do not, you must create a new agent, tool and/or task.
5. For each new agent, tool or task you should create a README.md file for human readers to understand what the agent does and how it does it. Each agent should have a test directory and test software to test the functionality of the agent, tools and/or tasks.  These can include a copy of prototype models to perform the test on.
6.  You can add to this file as you refine this process
