## Simulation Specification

The purpose of this document is to describe the analysis to be performed on the candidate aircraft

1.  Determine cruise speed at 75 % power
2.  Determine stall speed
3.  Perform an alpha sweep to determine performance as a function of alpha, including stability analysis
4.  Perform a beta sweep to determine performance as a function of beta, including stability analysis
5.  Determine underdamped,overdamped or unstable system response due to longitudianl pertubation
6.  Determinen underdamped, overdamped or unstable system response due to lateral pertubation
5.  Estimate aircraft mass and moments of inertia.  Use the weights specified in the SPECIFICATION.md document in the parent directory to assist with this.
6.  Compute aircraft total surface area
